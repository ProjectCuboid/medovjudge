## Medovjudge

**Medovjudge** is a repository containing solutions for various problems. Each solution includes a `.cpp` source file and a compiled binary.

### CONTENTS (Battaries Included)
- `sources/` – folder containing `.cpp` files.
- `builds/` - folder containing binaries. 
- `list.txt` – timestamps in the format `dd/mm/yy/HH/MM`.  
- `Makefile` – simplifies compilation and execution:  
  - Build a problem:  
    ```
    make all p="problemnum"
    ```  
  - Run a compiled problem:  
    ```
    make runall p="problemnum"
    ```
### NO AI!
 - No AI was/will be used in generating the solutions.
